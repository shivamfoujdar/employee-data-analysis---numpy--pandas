# employee-data-analysis---numpy--pandas
Pandas &amp; NumPy Employee Dataset Analysis with 50lakh Records
This project contains a **synthetic employee dataset with 5000000 records** and demonstrates various **data analysis techniques using Python's Pandas and NumPy libraries**.

## Dataset Overview

The dataset includes the following columns:

- `id` : Unique employee ID
- `age` : Employee age
- `salary` : Annual salary
- `experience_years` : Years of experience
- `department` : Department (IT, HR, Finance, Marketing, Sales)
- `performance_score` : Performance rating (1.0 - 5.0)
- `target` : Binary target variable (0 or 1)

## Features / Analysis Performed

- Viewing and inspecting the dataset
- Handling missing values
- Filtering data using multiple conditions
- Calculating mean, min, max, and aggregate statistics
- Grouping data by department and calculating average salary, performance, and count
- Sorting data in ascending/descending order
- Boolean indexing and creating new columns for feature engineering
- Exporting processed data to CSV

## Tools & Libraries

- Python 3.x
- Pandas
- NumPy
